| T-06-03-02 | Tampering | bypass suppression list via flat contactIds | mitigate | recipients-by-ids query applies the same suppression exclusion as the group path (MESG-09) |
| T-06-03-03 | Spoofing | empty-target campaign queued with 0 recipients | mitigate | create-time guard rejects when both groupIds and contactIds are empty |
