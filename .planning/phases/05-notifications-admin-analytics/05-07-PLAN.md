Build the admin/audit module (admin-service) providing the "full platform actions" audit log (ADMN-06)

cross-service coupling). D-14 — the admin/audit schema is its own small module. ADMN-06 — operators
view a filterable platform activity log.

a passive idempotent domain-event audit consumer; admin schema migrations.
